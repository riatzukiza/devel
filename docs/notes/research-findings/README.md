## Key Findings

- **Chinese (zh)**: Best covered — JailBench, MultiJail, JAILJUDGE
- **Japanese (ja)**: One solid dataset — APTO-001 (multi-turn)
- **German (de)**: Only ~79 prompts
- **Polish (pl)**: Zero datasets
- **Ukrainian (uk)**: Zero datasets
- **Russian (ru)**: Aya Red Teaming (subset)
- **Spanish (es)**: Aya Red Teaming (subset)

Attack mode gaps: Most modes (injection, obfuscation, tool abuse, code-mixing) have zero non-English coverage.
