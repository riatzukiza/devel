# Skill: OpenCode Tools, MCP, and ACP

## Goal
Implement or adjust OpenCode tools, MCP servers, and ACP integrations with correct protocols.

## Use This Skill When
- You add or modify tool definitions or execution behavior.
- You update MCP server configuration or ACP support.

## Do Not Use This Skill When
- The change is unrelated to tool calling or external integrations.

## Inputs
- Target tool or protocol changes.
- Existing OpenCode tool configuration and code.

## Steps
1. Validate tool behavior against OpenCode tools docs.
2. For MCP or ACP changes, verify protocol expectations with official docs.
3. Update any related configs or examples to match behavior.

## Output
- Updated tool/protocol behavior and aligned documentation.
