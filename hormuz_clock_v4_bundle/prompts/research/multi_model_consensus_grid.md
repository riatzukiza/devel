# Multi-Model Consensus Grid Prompt

Use this with multiple models (GPT, Claude, Perplexity, Gemini, etc.).
Each model should fill out the same grid:

- confirmed state changes since last run
- 0–4 scores for transit / attack / insurance / navigation / bypass / Asia buffers
- branch priors (reopening / effective closure / escalation)
- most important rewind trigger
- one design critique of the clock itself
- one new signal source worth integrating

Then compare outputs side by side to identify consensus, disagreement, and model bias.
