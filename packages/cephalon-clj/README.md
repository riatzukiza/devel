# promethean-jvm-clj (conversation export)

JVM Clojure skeleton reflecting the architecture we discussed:
- Always-on **Cephalon** loops (sessions/aspects)
- Reactive **Sentinel** contracts (e.g. note tagging)
- **Eidolon** semantic memory: embeddings + similarity search over seen memories/tool results

## Run
```bash
clj -M:run
```

## Env
- `OPENAI_BASE_URL`
- `OPENAI_API_KEY`
